# Agent Tools 快速使用指南

本项目包含 4 个独立的 AI 工具，每个工具都有自己的虚拟环境，互不干扰。

---

## 1. ASR (语音识别)

基于 Fun-ASR-Nano-2512 的语音识别工具。

### 安装环境（首次使用）

```bash
cd /Users/lifeng/git/git_agents/agent-tools/asr
bash setup_env.sh
```

### 使用（一行命令）

```bash
# 在任何目录下调用
/Users/lifeng/git/git_agents/agent-tools/asr/asr_recognize.sh 音频.wav 输出.txt

# 示例
/Users/lifeng/git/git_agents/agent-tools/asr/asr_recognize.sh test.wav result.txt
/Users/lifeng/git/git_agents/agent-tools/asr/asr_recognize.sh test.wav result.txt 英文
```

### 支持语言

- 中文（默认）
- 英文
- 日文

---

## 2. TTS (语音合成)

基于 Qwen3-TTS-12Hz-1.7B-CustomVoice 的文本转语音工具。

### 安装环境（首次使用）

```bash
cd /Users/lifeng/git/git_agents/agent-tools/tts
bash setup_env.sh
```

### 使用（一行命令）

```bash
# 在任何目录下调用
/Users/lifeng/git/git_agents/agent-tools/tts/tts_speak.sh "要说的话" 输出.wav

# 示例
/Users/lifeng/git/git_agents/agent-tools/tts/tts_speak.sh "你好世界" output.wav
/Users/lifeng/git/git_agents/agent-tools/tts/tts_speak.sh "Hello" output.wav English
```

### 支持语言

- Chinese (中文，默认)
- English (英文)

### 支持音色

- Vivian (默认)
- 更多音色请参考 tts/README.md

## 3. DeepSeek-OCR (图像识别)

基于 DeepSeek-OCR-2 的图像文字识别工具。

### 安装环境（首次使用）

```bash
cd /Users/lifeng/git/git_agents/agent-tools/deepseek_ocr
bash fix_deepseek_ocr.sh
```

### 使用（一行命令）

```bash
# 在任何目录下调用
/Users/lifeng/git/git_agents/agent-tools/deepseek_ocr/ocr_image.sh 图片.png 输出.md

# 示例
/Users/lifeng/git/git_agents/agent-tools/deepseek_ocr/ocr_image.sh test.png result.md
```

### 输出格式

- Markdown 格式，自动清理标签

## 优点

✅ **独立环境** - 每个工具都有自己的虚拟环境，版本冲突完全隔离  
✅ **一行调用** - 不需要手动激活虚拟环境  
✅ **任意目录** - 在任何地方都能调用  
✅ **自动管理** - 脚本自动激活和退出环境
